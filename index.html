Technology Stack

Engineered for Scale

Modern AI and cloud infrastructure built to serve millions of farmers across India.

Frontend

  • Vanilla JS + GSAP scroll animations
  • Chart.js for data visualisation

  • Firebase Firestore SDK v9

AI / ML

  • TensorFlow + MobileNetV2

  • Transfer learning + fine-tuning
  • Model hosted on Hugging Face Hub

Backend

  • FastAPI with async routes

  • Deployed on Render
